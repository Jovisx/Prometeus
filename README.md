# Prometeus
All you need in a markdown editor

An editor created for everyone simple intuitive and fast.


 ## How to use
 
Clone the repo
`cd to Prometeus ` 

Install
`npm install `

Run
`npm start `

# Goal
My goal is to create a store where each developer can create their own sites and sell them.
